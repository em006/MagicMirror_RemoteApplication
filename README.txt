The Mobile App folder contains the code for the React Native app. It can be run using expo.
The Modules folder contains our custom MagicMirror modules. They should be downloaded and placed into the modules folder of a MagicMirror installation following the instructions on magicmirror.builders
The HTTP Server folder contains the code for the Node.js RESTful API server.
The MQTT Broker folder contains the Python code for the MQTT server.
